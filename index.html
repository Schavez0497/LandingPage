Mis Habilidades

A continuación se presentan algunas de las tecnologías con las que trabajo:

DOCKER

Docker es una plataforma de código abierto que permite desarrollar, enviar y ejecutar aplicaciones dentro de contenedores. Facilita la creación de entornos aislados y portátiles, asegurando consistencia desde el desarrollo hasta la producción.

KUBERNETES

Kubernetes es una plataforma de código abierto diseñada para automatizar la implementación, escalado y operación de aplicaciones en contenedores, gestionando clústeres de hosts a través de un sistema extensible y portátil.

HELM

Helm es un gestor de paquetes para Kubernetes que facilita la definición, instalación y actualización de aplicaciones en clústeres. Utiliza "charts" para empaquetar recursos Kubernetes, simplificando despliegues y gestión.

Bootstrap

Bootstrap es un marco de trabajo de código abierto para desarrollar sitios web y aplicaciones web responsivas y móviles. Proporciona plantillas de diseño basadas en HTML, CSS y JavaScript para facilitar el desarrollo.

JAVA

Java es un lenguaje de programación de alto nivel, orientado a objetos y con enfoque en portabilidad. Permite desarrollar aplicaciones multiplataforma gracias a su máquina virtual (JVM) que ejecuta el código en cualquier sistema operativo

HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear y estructurar páginas web. Utiliza etiquetas para definir elementos como texto, imágenes, enlaces y formularios, formando la base de cualquier sitio web.
